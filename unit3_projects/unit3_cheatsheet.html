Cheat Sheet: Looping through Arrays with Ruby & Javascript

So you want to perform some operation on each item in an array?

Below is a list of ways I most often iterate through an array with Ruby & Javascript.

The examples below refer to the following sample arrays:

  1. students = ["Farheen", "Joel", "Maria", "Tanner", "Sam"]
  2. numbers = [1,2,3,4,5]

    • Ruby

    #collect

    Invokes the given block once for each element of self. Creates a new array containing the values returned by the block. Can be descructive if written as collect!. Same as #map

    1. students.collect { |student| p student +"!"}

    #each

    Calls the given block once for each element in self, passing that element as a parameter

    1. numbers.each {|number| p number}

    #each_index

    Same as #each, but passes the index of the element instead of the element itself.

    1. numbers.each_index {|i| p i}

    #delete_if

    Deletes every element of self for which block evaluates to true. The array is changed instantly every time the block is called, not after the iteration is over.

    1. p students.delete_if {|name| name.length > 6 }

    #find_index

    Returns the index of the first object in ary such that the object is == to obj. If a block is given instead of an argument, returns the index of the first object for which the block returns true. Returns nil if no match is found.

    1. p students.find_index("Joel")
    2. p numbers.find_index(5)

    for..in

    Executes codeonce for each element in expression.

    1. for n in numbers
    2.   p n+= 1
    3. end

    #map

    Invokes the given block once for each element of self. Creates a new array containing the values returned by the block. Can be destructive if written as map!. Same as #collect.

    1. students.map { |student| p student.reverse.split("") }

    #select

    Returns a new array containing all elements of ary for which the given block returns a true value.

    1. p numbers.select{ |number| number< 4}
