Cheat Sheet: Looping through Arrays with Ruby & Javascript

The loop examples below refer to the following arrays:
students = ["Farheen", "Joel", "Maria", "Tanner", "Sam", "Dylan", "Justin"]
numbers = [1,2,3,4,5]

Ruby

EACH

numbers.each {|number| p number}

FOR

for n in numbers
p n+= 1
end

MAP

students.map { |student| p student.reverse.split("") }

can also be destructive: .map!

COLLECT

students.collect { |student| p student +"!"}

DELETE_IF

p students.delete_if {|name| name.length > 6 }

FIND_INDEX

p students.find_index("Joel")
p numbers.find_index(5)

EACH_INDEX

numbers.each_index {|i| p i}

SELECT

p numbers.select{ |number| number < 4}
