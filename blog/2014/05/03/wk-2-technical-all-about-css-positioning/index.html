All About CSS Positioning

May 3rd, 2014

This blog post will discuss the differences between relative, absolute, and fixed positioning.

The positionproperty is important for making more complex web page layouts. Possible values for this property include:

  • static
  • relative
  • absolute
  • fixed

Static Positioning

Static positioning is the default position value for any element on a website. Elements with this positioning or no positioning will fall where they are found in the standard flow of a document. These elements scroll normally.

Relative positioning means an element will fall relative to their origin or starting point. Elements with this positioning follow the flow of the document. These elements scroll normally.

Absolute Positioning

Absolute positioning can remove an element from the flow of the document, allowing you to position an item exactly where you want it. Elements with this positioning are not affected by other elements or by the standard flow of the document. These elements scroll normally.

Fixed Positioning

Like absolute positioning, fixed positioning allows you to position an element exactly where you want it and can remove an element from the flow of the document. These elements will not scroll normally.
