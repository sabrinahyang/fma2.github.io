Thursday, May 8, 2014

Why is JavaScript so popular in web development?

JavaScript is popular in web development because it is arguably the best language for the web. Here are the top 4 signs it is the best language for the web--

  1. It is powerful, supporting both client-side and server-side development--the latter increasingly so.


    2. (Javascript != above image)

    Though Javascript has been around for a long time, previously, JavaScript was limited to front end development--web pages, User Interfaces and fetching data, etc. With Node.js and Google's V8 JavaScript Engine, this is quickly changing to more server-side development. Effectively, "JavaScript is quickly becoming not just the ideal language for the web, but indeed a big composition of the web itself." It continues to show that is a capable language, especially in "ever-growing number of cases where asynchronous communications are necessary."1

  2. It has the potential to maximize the user experience.

    3. User experience (UX) refers to a person's attitude and behavior towards a product, like a website or application. Apple raised the bar for what is considered acceptable in terms of user experience--the "success and adoption of an app is highly correlated"2 to this. JavaScript has the potential to deliver the necessary "rich, dynamic web content" 2 that attends to user experience.

  3. It is supported by virtually all browsers and adapts to multiple devices.

    4. Effectively, "millions of people run browsers that can handle JavaScript--in some variation or another."3 It is standardized across common browsers. Further, most devices--tablets, e-readers and smart phones--have javascript language. Javascript is especially useful for devices with Apps.

  4. It is easy to use.

    5. One can add JavaScript to his/her site without "having access to a server, compiling programs, or having the browser load a virtual machine."3 Running JavaScript is as simple as adding it to a web page and uploading the page. This is significant considering the multitude of user experience features JavaScript can add to a web page. Further, it is easy to pick up because of "the very flexible nature of the language."1
